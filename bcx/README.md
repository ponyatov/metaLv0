#  `bcx`
## Byte Code eXecution engine

Scripting & CLI engine: FORTH shell

* stack-based bytecode interpreter /ANSI C/
* targets for tiny embedded systems
  * microcontrollers
  * embedded Linux
  * tiny games
* powered by `metaL`
